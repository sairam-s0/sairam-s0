## Projects

### ActiveLabelingSystem
An active learning pipeline that keeps a human in the loop while training runs in the background — handles labeling, training, and model promotion without downtime.
- Human-in-the-loop labeling
- Confidence/entropy-based sample selection
- Background shadow training with zero-downtime promotion

**Stack:** Python · PyQt6 · YOLOv8 · Ray · SQLite
[Repo →](https://github.com/sairam-s0/ActiveLabelingSystem)

### Local AI Automation
Captures a screenshot, pulls text out via OCR, and runs it through a local LLM. No cloud, no API keys, no external calls.
- Screenshot-based input capture
- OCR text extraction
- Local inference with Phi-4 Mini via Ollama

**Stack:** Python · Phi-4 Mini · OCR · Ollama
[Repo →](https://github.com/sairam-s0/local_ai_automation)

### Personal AI Assistant
A local-first personal assistant with a FastAPI backend, Groq for chat completions, SQLite for storage, and a lightweight browser UI. Handles conversation memory, tasks, reminders, and optional Google Calendar integration.
- Browser chat UI with a `/chat` API backed by Groq
- Persistent chat sessions, memories, tasks, and reminders in SQLite
- Tool-calling for memory, tasks, reminders, current time, and Google Calendar
- Google Calendar OAuth login with calendar/event operations
- Test coverage for the API, tools, prompt building, and Google auth flow

**Stack:** Python · FastAPI · Groq SDK · SQLite · Pydantic Settings · Google Calendar API · Pytest
[Repo →](https://github.com/sairam-s0/personal-ai-assistant)

### Ponzi Scheme Detection
A team project that uses ML to flag potentially fraudulent financial transaction patterns.
- Transaction-level analysis
- Anomaly detection on financial data

**Stack:** Python · Scikit-learn · Pandas
[Repo →](https://github.com/Vijay-31-08-2005/ponzi-scheme-detection)

### Metadata Editor Pro
A small utility for viewing and cleaning metadata from images and documents — useful for privacy and basic file hygiene.
- Works on images and documents

**Stack:** Python · Tkinter · Pillow · piexif
